# Runbook Validator

Valida que los Runbooks cumplan las reglas DOC-001, DOC-002, DOC-004, DOC-006.

## Reglas aplicadas

- `DOC-001`: front-matter completo
- `DOC-002`: valor de status válido
- `DOC-004`: registrado en manifest.json
- `DOC-006`: secciones obligatorias (Síntomas, Diagnóstico, Procedimientos, Rollback)
