# ADR Validator

Valida que los ADRs cumplan las reglas DOC-001, DOC-002, DOC-003, DOC-004, DOC-007.

## Reglas aplicadas

- `DOC-001`: front-matter completo
- `DOC-002`: valor de status válido
- `DOC-003`: naming secuencial `adr-NNN-slug.md`
- `DOC-004`: registrado en manifest.json
- `DOC-007`: secciones obligatorias (Contexto, Decisión, Alternativas Consideradas, Consecuencias)
